# TMC Warehouse System (скелет проекта)

Этот репозиторий содержит каркас системы приёма ТМЦ на склад:

- клиентское десктоп-приложение на PySide6;
- локальный сервер на FastAPI + SQLite + Peewee;
- общие Pydantic-модели в `common/`;
- документацию в `docs/`.

Основные документы:

- `docs/ARCHITECTURE.md` — архитектура и флоу.
- `docs/API.md` — спецификация REST API.
- `docs/INSTALL.md` — установка и запуск.
- `docs/USER_GUIDE.md` — руководство пользователя.

Конфигурация: `config/config.json`.

Зависимости: `requirements.txt`.

Дальнейшая реализация кода (client/server) должна строго следовать архитектуре и моделям, описанным в документации.
